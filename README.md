# SecurityFoscusOnline2023課程推動模式與活動
- 模式一:周末資安行
  - 只要一個星期六與星期日完整上完A1-A3課程模組
  - 如果全部都沒學過兩天要完成~是有點硬
- 模式二:快樂學資安
  - 分開舉辦:兩個晚上(如星期二與四晚上3小時) + 一個星期六或日即可
  - A2_Linux資安技術入門~可以實體上課
